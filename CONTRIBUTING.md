## ğŸ¤” How Can I Contribute?

### Reporting Bugs

Before creating bug reports, please check the [existing issues](https://github.com/Rainbow-CPaaS/Rainbow-React-Native-Samples/issues) to see if the problem has already been reported.

When creating a bug report, please include:
- A clear, descriptive title
- Steps to reproduce the issue
- Expected vs. actual behavior
- Screenshots or screen recordings if applicable
- Device/OS version
- React Native version
- Rainbow SDK version

### Suggesting Enhancements

We welcome enhancement suggestions that improve the sample app. Please include:
- A clear, descriptive title
- A description of the suggested enhancement
- Why this enhancement would be useful
- Any potential drawbacks or alternatives

### Code Contributions

1. **Fork** the repository on GitHub
2. **Clone** your fork locally
3. **Create a new branch** for your changes
4. **Commit** your changes with clear, descriptive messages
5. **Push** your branch to your fork
6. Open a **Pull Request**

## ğŸ›  Development Setup

1. **Prerequisites**
   - Node.js (v18+)
   - Yarn or npm
   - React Native development environment
   - Android Studio / Xcode

2. **Install Dependencies**
   ```bash
   yarn install
   # or
   npm install
   ```

3. **Run the App**
   ```bash
   # Android
   yarn android
   
   # iOS
   cd ios && pod install && cd ..
   yarn ios
   ```
